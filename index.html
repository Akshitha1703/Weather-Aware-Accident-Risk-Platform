Project Overview

The Weather-Aware Accident Risk Platform is an interactive analytical solution built using Power BI, integrating U.S. accident data with weather indicators to understand how environmental conditions influence accident frequency and severity.

Data Source

Primary dataset: U.S. Accidents — Kaggle (2016–2023)

Includes road conditions, severity, geolocation, weather variables, and timestamps.

Data Pipeline

The end-to-end pipeline includes:

  • Dataset cleaning in Power Query
  • Splitting into Accident_Data & Weather_Data
  • SharePoint hosted auto-refresh
  • Modeling with star schema
  • DAX calculations for KPIs

How to Use This Report

1. Use slicers at the top to filter by:

  • State / City
  • Year
  • Weather Condition
  • Month / Weekday / Hour

2. Hover or click visuals to view tooltips, cross-highlight trends, and compare values.

3. Navigate using page tabs (Overview, Weather Impact, Location Risk, Trends).

4. Insights text is provided under visuals to explain patterns clearly.

5. Use the map to identify high-risk regions and weather patterns.
