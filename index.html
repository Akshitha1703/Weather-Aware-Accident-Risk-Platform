Data Source

ğŸ“¦ Kaggle Dataset (Updated)

The project uses the US Accidents dataset: accident ID & severity, geolocation (city/state/ZIP), timestamps, plus weather metrics (temperature, humidity, pressure, visibility, wind, precipitation).

Kept for Analysis

  • Accident: ID, Severity, Start/End Time, Distance, City/County/State, Zip, Timezone
  • Weather: Timestamp, Temperature(F), Humidity(%), Pressure(in), Visibility(mi), Wind Speed(mph), Precipitation(in), Condition

Removed to Focus
